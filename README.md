# Harisrujan C – Portfolio

AI Product Architect & GenAI Mentor portfolio site.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS
- Framer Motion

## Local Development

```sh
# Install dependencies
npm install

# Start dev server
npm run dev
```

The app runs at [http://localhost:8080](http://localhost:8080).

## Build

```sh
npm run build
```
